Example 3.5.

The following sets are also vector spaces:

  1. The set of all polynomials (in variable \(t\)) denoted \(\mathbb{P}\text{.}\) %\item \((\mathbb{R})^\mathbb{Z}\text{,}\) the set of bi-infinite sequences of real numbers. If \(x \in (\mathbb{R})^\mathbb{Z}\text{,}\) then \(x=... x_{-3},x_{-2},x_{-1}, x_0,x_1,x_2,x_3,...\) where \(x_i \in \mathbb{R}\text{.}\) Scalar multiplication and vector addition are done coordinate-wise in this vector space, i.e.

    \begin{equation*} x+y = ..., x_{-3}+y_{-3} , x_{-2}+y_{-2} , x_{-1}+y_{-1} , x_0+y_0 , x_1+y_1 , x_2+x_2 , x_3+y_3,... \end{equation*}

  2. \(F(S,\mathbb{R})\text{,}\) the set of functions from a set \(S\) to the real numbers. We will take a closer look at this vector space in the next problem.

  3. \(\{\vec{0}\}\text{,}\) the trivial vector space.
