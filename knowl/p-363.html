If \(T\) is a linear transformation from \(\mathbb{R}^n\) to \(\mathbb{R}^m\text{,}\) then the standard matrix presentation of \(T\) is a \(m\) by \(n\) matrix

\begin{equation*} A=[T(\vec{e_1}) \quad T(\vec{e_2}) \quad ... \quad T(\vec{e_n}) ] \end{equation*}

where \(\vec{e_i}\) is the \(i\)-th elementary vector of \(\R^n\text{.}\) Note that \((\vec{e_i})_j = \delta_{i,j}\text{,}\) where \(\delta\) is the Dirac delta function defined by

\begin{equation*} \delta_{i,j}=\left\{ \begin{array}{cc} 0 \amp \mbox{if }i\neq j\\ 1 \amp \mbox{if } i = j \end{array} \right. \end{equation*}
